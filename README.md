# pizza_sales-dashboard
1.Pizza sales analytics:Pizza Sales Report Insights Dashboard
This project presents an interactive Pizza Sales Dashboard built using Power BI to analyze sales performance, customer ordering behavior, and product trends. The dashboard transforms raw transactional data into meaningful insights that help in understanding revenue drivers, popular products, and operational patterns.
The goal of this project is to demonstrate data analysis, visualization, and storytelling skills by uncovering key business insights from pizza sales data.
2.Tools & Technologies Used
Power BI – Data visualization and dashboard creation
Microsoft Excel / SQL – Data cleaning, preprocessing, and transformation
DAX (Data Analysis Expressions) – Creating calculated measures and KPIs

3.Key Performance Indicators (KPIs)

The dashboard highlights the following important metrics:

Total Revenue: 817.86K
Average Order Value: 38.31
Total Pizzas Sold: 49,574
Total Orders: 21,350
Average Pizzas per Order: 2.32

These KPIs provide a quick snapshot of overall business performance.

4.📈 Dashboard Features & Insights
📅 Order Trends Analysis
Orders by Day:
Sales gradually increase throughout the week.
Peak order days are Thursday, Friday, and Saturday, indicating higher customer activity towards weekends.
Orders by Month:
Highest order volumes are observed in July, January, and March.
Seasonal trends suggest fluctuations in customer demand across different months.


5.🍕 Sales Distribution
🔹 Revenue by Pizza Category
Categories include Classic, Supreme, Chicken, and Veggie.
Revenue contribution is fairly balanced across categories, with Classic pizzas contributing the highest share.
🔹 Revenue by Pizza Size
Large-sized pizzas contribute the most revenue (~45.89%).
Medium and regular sizes follow, while extra-large sizes have minimal contribution.
Indicates strong customer preference for larger portion sizes.

6.📦 Product Performance
🔹 Total Pizza Sold by Category
Classic pizzas lead in total sales (~15K units).
Supreme and Veggie categories show similar performance (~12K each).
Chicken category has slightly lower sales (~11K).
🔹 Top Revenue-Generating Pizzas
The highest revenue is generated by:
Thai Chicken Pizza
Barbecue Chicken Pizza
California Chicken Pizza
Classic Deluxe Pizza

These items are key contributors to overall profitability.

7.🔍 Key Business Insights
📌 Peak Sales Periods: July, January, and March show maximum order volume.
📌 Weekend Demand: Orders significantly increase towards the weekend (Thu–Sat).
📌 Top Category: Classic pizzas dominate both revenue and quantity sold.
📌 Customer Preference: Large-sized pizzas are the most popular choice.
📌 High-Performing Products: Chicken-based pizzas generate strong revenu

8.🎯 Project Objectives Achieved
Cleaned and transformed raw sales data into structured format
Built an interactive dashboard with slicers (pizza category & month filters)

Delivered actionable insights for business decision-making

9.🚀 Conclusion

This dashboard provides a comprehensive view of pizza sales performance and helps stakeholders make data-driven decisions. It highlights customer preferences, seasonal demand patterns, and top-performing products, which can be used to optimize marketing strategies, inventory planning, and pricing.
